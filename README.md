# Hanna's Bridesmaid Dress Matcher

A web app to help match bridesmaid dress colors to our Lake Como wedding palette.

## Features

- **Interactive Color Selection**: Click 3 points on any dress photo to sample the exact color
- **Smart Matching Algorithm**: Uses perceptual color distance to find the closest match
- **Mediterranean Palette**: 7 warm, earthy tones perfect for Lake Como in August
- **Works Entirely in Browser**: No uploads to servers - all processing happens locally

## How to Use

1. Upload a photo of a bridesmaid dress
2. Click on 3 different spots on the dress fabric to sample the color
3. Click "Analyze Color" to see if it matches the wedding palette
4. Get instant feedback on whether the dress is a perfect match!

## Color Palette

- Golden Olive
- Warm Khaki
- Dusty Sage
- Moss Green
- Earthy Olive
- Olive Drab
- Soft Golden Sage

## Technology

Built with vanilla HTML, CSS, and JavaScript using the Canvas API for color extraction.
